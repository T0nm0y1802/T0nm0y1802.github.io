Project Title: YOLO Automatic License Plate Detection

Data Collection & Preprocessing
- Gathered a diverse dataset of license plate images.
- Preprocessed images for uniformity and clarity.

Model Training
- Utilized the YOLO (You Only Look Once) object detection algorithm for real-time license plate detection.
- Trained the model with a comprehensive dataset to ensure high accuracy.

Optical Character Recognition (OCR)
- Integrated Tesseract OCR for extracting text from detected license plates.
- Fine-tuned OCR settings for better accuracy in text recognition.

Model Deployment
- Deployed the system using a user-friendly Streamlit web application.
- Enabled users to upload images or provide video links for automatic license plate detection and text extraction.

Web Application
- Developed a Streamlit web app for easy user interaction.
- Provided real-time feedback and results for license plate detection and text extraction.
